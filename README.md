# Formulário Multi-Etapas

Formulário de cadastro multi-etapas 📝

# Tecnologia

- React JS
- TypeScript

# Começando
- Dependências:
  - styled-components
    - npm install styled-components
    - npm install -D @types/styled-components
  - react router dom
    - npm install react-router-dom
    - npm install -D @types/react-router-dom
